# React-Router tutorial repo (from official React Router documentation)
As the name says.
Basically React Router is useful because you can implement SPA (Single-Page Application) with it. It's like when you click on a link or a button, you get the content, but your page doesn't refresh or loads new page. Let's say if YouTube wasn't SPA then whenever you try to open a video your page will refresh or worse - whenever you post a comment your page refreshes. SPA fixes that. Also makes routing easier since you don't need to implement routing on your backend, like manually creating a function that returns HTML file of each page.
